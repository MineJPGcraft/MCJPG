# MCJPG Organization Rules and Regulations

1. The **MCJPG** Server Exchange Organization (hereinafter referred to as "the Organization" or "MCJPG Organization") is a coordinated entity formed by server hosts such as 随风潜入夜 FireGuo 二氧化钛 Weasd-QAQ among others, for mutual development.
2. Joining the Organization is deemed as agreeing to the following disclaimer.

## Disclaimer
1. The Organization aims to provide support, promotion, and technical assistance to servers in need.
2. The suggestions provided by the Organization are not professional opinions and are for reference only. The Organization does not bear any responsibility for any adverse consequences resulting from following the Organization's suggestions.
3. During crowdfunding events held by the Organization, server hosts can choose whether to participate. These activities are generally not mandatory, except in special cases.
4. The Organization does not force server hosts to join or leave. Joining the Organization is considered voluntary, and leaving is also deemed voluntary. The Organization collects server addresses, ports, and other materials solely for review, status monitoring, and providing suitable players with access, with no other illegal purposes. (The Organization merely acts as a conduit in providing server-related information to players.)
5. Upon leaving the Organization, unless in special situations (such as violation of the Organization's regulations), it is considered voluntary withdrawal.
6. After exiting the Organization, the Organization will delete the "servernamexxx.mcjpg.org" domain provided to the exiting server, and any resulting consequences must be assumed by the leaver.
7. The Organization reserves the right of final interpretation and the right to edit this disclaimer at any time. If you have any objections to this disclaimer, please contact the Organization's management team via private message.
>
> **Joining the Organization implies that you have read and agree to this disclaimer.**
>

## Joining the Organization

### Participating in Organizational Activities

1. New members must undergo a review by community group auditors. Disagreeing with the review implies forfeiting the opportunity to join the Organization.
2. After joining, one must adhere to the relevant regulations and actively fulfill corresponding obligations. Violators will face various degrees of punishment, with severe cases resulting in expulsion from the Organization and permanent prohibition from joining.
3. Successfully joining the Organization's server owner group designates the server as a member server of the Organization.

### Promotion Regulations

1. Member servers must state their affiliation with MCJPG Organization on their promotional webpages, with a correct link to https://mcjpg.org/ (ignore if there is no promotional webpage).
2. For the healthy development of MCJPG Organization, we need to collect sufficient information to identify and resolve issues. Therefore, it is recommended that member servers provide reasons when exiting MCJPG Organization.
3. MCJPG Organization allows and supports servers conducting activities in its name and independently organizing activities (including but not limited to promotional events and competitions).
4. **Service Provider Promotion** **Disclaimer**: This promotional notice aims to provide general information regarding service provider promotion and does not constitute professional advice of any kind; it is for reference only. The Organization offers effective server solutions to its members but accepts no liability for any direct or indirect losses arising from the use or reliance on this information. Furthermore, the management has not entered into any benefit-sharing agreement with service providers. The Organization reserves the final right of interpretation of this disclaimer.
If you have objections, please send feedback to the Organization's leader via private message.



## How to Add Your Server Display Position on the Organization's Website?

###  Provide necessary information about the server, including:

  1. Server Name
  2. Server Icon (optional)
  3. Server Information
  4. Server Description
  5. Server Version
  6. Promotion Link (can be a joining link for QQ groups, server promotional website link, or promotional video link)

###  Send an application containing the necessary information to the leader

``` txt
Server Name:
Server Icon:
Server Information:
Server Description:
Server Version:
Promotion Link:
```

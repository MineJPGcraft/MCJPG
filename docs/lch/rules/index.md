# MCJPG 組織規章制度

## 📋 組織簡介

**MCJPG** 伺服器交流組織(以下簡稱「本組織」、「集宣組織」)，乃由**隨風潛入夜**、**FireGuo**、**二氧化鈦**、**Weasd-QAQ**等諸位伺服器服主之為共謀發展，而聯袂組建之有組織之整體也。

:::warning 重要提示
凡入本組織者，即視為同意下列免責聲明及全部規章制度。
:::

---

## ⚖️ 免責聲明

1. **組織宗旨**
   本組織之宗旨，在於對各有所需之伺服器，提供相應之助扶、宣傳推廣及技術支援等。

2. **建議參酌性**
   本組織所提建議，不具專業性，僅供參酌之用。遵循本組織所提建議時，如造成不良後果，本組織不負任何責任。

3. **活動自願性**
   本組織舉行眾籌活動時，各伺服器服主可自行決定是否參與，除特殊情況外，此類活動不具強制性。

4. **自願加入及退出**
   本組織不強迫各伺服器服主加入或退出，加入本組織者視為自願加入，退出者視為自願退出。本組織所收集之各伺服器地址、端口等資料，僅作為審核、添加狀態監測及提供給適合玩家遊玩之用，無其他之、非法之目的(對於向玩家提供之伺服器相關訊息，本組織僅起傳達作用)。

5. **退出處理**
   在退出本組織時，除特殊情況外(如違反本組織相關規定)，均視為自願退出。

6. **域名回收政策**
   退出本組織後，本組織將刪除為退出伺服器所提供之如「servernamexxx.mcjpg.dev」域名，所造成之後果由退出者自行承擔。

7. **最終解釋權**
   本組織保留對本免責聲明之最終解釋權及隨時進行編輯之權利。如君對本免責聲明有任何異議，請即時私信聯繫本組織管理組。

:::danger 特別聲明
**凡加入本組織者，視為已閱讀完畢並同意本免責聲明。**
:::

## 🚪 加入本組織

### 📝 參與本組織活動

1. **審核流程**
   欲入本組織者，須受社群團隊內審核員之審查，如不願受審，即視為捨棄入組之念。

2. **遵守規定**
   既入本組織，當遵相關規章制度，並力行相應之義務。違規者，將受不同程度之罰，情節重者，逐出本組織，永禁加入。

3. **成員認定**
   服主成功入本組織服主群，即視其伺服器為本組織成員伺服器。

---

### 📢 宣傳條例

1. **標識要求**
   成員伺服器須於MCJPG宣傳網頁中標明集宣組織之名，並設置正確之指向鏈接[https://mcjpg.org/](https://mcjpg.org/)
   (若無宣傳網頁，則此項可略)

2. **退出反饋**
   為集宣組織之良性發展，吾等需搜集充足之資訊，以發現並解決潛在之問題。故建議各成員伺服器於退出集宣組織時，須提出退出集宣組織之緣由。

3. **活動支援**
   集宣組織允許並支援各伺服器以集宣組織之名義舉辦活動，同時亦允許並支援各伺服器自由自主地組織活動(活動包括但不限於宣傳活動、評比活動等)。

4. **服務商宣傳免責聲明**
   本宣傳公告旨在提供有關服務商宣傳之一般資訊，並非專業建議之任何形式，僅供參考之用。本組織旨在為各成員提供更有效之伺服器解決方案，但對於因使用或依賴此等資訊而產生之直接或間接損失，不負任何責任。且本組織管理層未與任何服務商簽署任何利益分配協議。本組織保留對本免責聲明之最終解釋權。
   如君對此有異議，請即時向本組織負責人私信反饋。

## 🎮 如何在組織官網中添加己之伺服器？

### 步驟一：準備伺服器訊息

請提供以下必要之資訊：

| 訊息項 | 說明 | 是否必需 |
|--------|------|----------|
| 伺服器之名 | 君之伺服器名稱 | ✅ 必需 |
| 伺服器圖標 | 伺服器之標識圖標 | ⭕ 可選 |
| 伺服器訊息 | 伺服器之基本訊息(如IP地址、端口等) | ✅ 必需 |
| 伺服器描述 | 伺服器之詳細介紹 | ✅ 必需 |
| 伺服器版本 | 所支援之Minecraft版本 | ✅ 必需 |
| 宣傳鏈接 | QQ群鏈接、官網鏈接或宣傳視頻之鏈接 | ✅ 必需 |

---

### 步驟二：提交申請

致書於負責之人，呈上如下格式之申請訊息：

```text
伺服器之名：
伺服器圖標：
伺服器訊息：
伺服器描述：
伺服器版本：
宣傳鏈接：
```

---

### 步驟三：待審核結果

審核通過後，君之伺服器將現於MCJPG官網之伺服器列表之中。

:::tip 💡 高級選項
若君有一GitHub賬戶，可直照網站倉庫之說明提交『Pull Request』，助吾輩更迅速地更新伺服器之列表！

🔗 吾輩之GitHub倉庫：[https://github.com/MineJPGcraft/MCJPG](https://github.com/MineJPGcraft/MCJPG)
:::

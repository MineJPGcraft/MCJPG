# MCJPG Server Communication Organization

Welcome to the Minecraft Server Communication Organization (MCJPG)!

An organization dedicated to Minecraft server technology exchange and promotion.

[Join the MCJPG Community Group](https://qm.qq.com/q/5Y4ueZdkxq)

## Submit a Server

Before submitting a server, please ensure your server meets the following requirements:

- The server is operational and will not shut down in the short term.

- The server has joined the **MCJPG** organization.

If your server meets the above requirements, please go to the [Server List Repository](https://github.com/MineJPGcraft/Serverlist) and submit a Pull Request following the repository instructions.
